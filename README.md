# A Jekyll Boilerplate 

## About
### Features

- Jekyll
- Bootstrap V4 (alpha)
- Sass
- Gulp


### Future Features

- CSS Minification
- JS Minification
- Image Compression

## Quickstart

### Setup

`npm install`

### Start Local Server

`gulp`


### Production

Use the files under `\_site\`
